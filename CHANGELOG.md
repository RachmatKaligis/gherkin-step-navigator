# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-30

### Added
- **Go to Definition** — Ctrl+Click on Gherkin steps to navigate to Robot Framework keyword definitions
- **Hover Information** — Hover over steps to see keyword signature, documentation, arguments, and location
- **Smart Matching** — Exact matching and pattern matching for keywords with `${variable}` placeholders
- **Keyword Caching** — 30-second cache for fast repeated lookups
- Support for `.resource` files in `steps/` directories and workspace-wide
- Support for both `gherkin` and `markdown` language IDs (for `.feature.md` files)
